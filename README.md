# Polydicer

A web version of Polydice business card.

## Setup

* Install required gems:

```
bundle install
```

* Set up your info in `me.json`

```
bundle exec rake init
```

* Start up a local web server at `http://localhost:4567/`

```
bundle exec middleman server
```
